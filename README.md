# ForzaForza

* go in `CXGame1.0/connectx/`
* compile with `javac -cp ".." ForzaForza/*.java`

* test `java -cp ".." connectx.CXPlayerTester 6 7 4 connectx.L1.L1 connectx.ForzaForza.ForzaForza -v -t 1 -r 5`
